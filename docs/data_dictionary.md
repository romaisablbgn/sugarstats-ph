# Data Dictionary

## product_type

- raw_sugar
- washed_sugar
- refined_sugar
- molasses

## sugar_class

- "A" = U.S. quota export sugar
- "B" = domestic sugar
- "C" = reserve sugar
- "D" = world market export / surplus export sugar
- "E" = sugar for export-oriented food processors/manufacturers

## units

- MT = metric tons
- LKg = 50-kg bags
- PHP/LKg = pesos per 50-kg bag
- PHP/kg = pesos per kilo
- PHP/MT = pesos per metric ton

## survey_status

- reported = numeric value exists
- missing = blank or dash
- terminated = price survey was terminated/not conducted
- unparsed = value could not be safely interpreted
